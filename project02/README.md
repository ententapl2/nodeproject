# Opis

### Umfragen to prosty serwis internetowy do publikacji ankiet, które dzięki możliwości zakładania kont są dostępne do głosowania dla innych osób.

# Instalacja

1. Sklonuj repozytorium do katalogu ``` git clone https://github.com/ententapl2/nodeproject.git ```
2. Wejdź w katalog project02 ``` cd nodeproject/project02/```
3. Zainstaluj zależności ``` npm i ```
4. Uruchom serwer poprzez ``` node src/index.js ```
